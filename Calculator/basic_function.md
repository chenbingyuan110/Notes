### `JUMP` Jump to

- `TOP` Jump to the first line
- `BTM` Jump to the last line
- `PgUp` PageUp
- `PgDn` PageDown

### `DEL` Delete

- `DEL-L` Delete this line

- `DEL-A` Delete all
